COMMENT JOUER

1 - Faite un npm install si le dossier node_modules n'est pas présent

2 - Faite un npm start dans le dossier du jeu ou un npm index.js dans un invité de commande

3 - Connectez-vous à l'adresse localhost:5000

4 - Le premier joueur créer une nouvelle partie en entrant son pseudo

5 - Le deuxième joueur (dans un second onglet toujours à l'adresse localhost:5000) entre son propre pseudo ainsi que l'identifiant du salon de jeu affiché sur la fenêtre du premier joueur

6 - Cliquer ensuite sur rejoindre la partie
